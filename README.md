# template-vike-elysia-solid-daisyui

Template stack:
- **Vike**
- **Elysia**
- **Vite**
- **Solid**
- **DaisyUI**
- **ESLint**
- **TypeScript**

## Clone
```sh
git clone https://github.com/rtritto/template-vike-elysia-solid-daisyui.git
```

## Install
```sh
yarn
```

## Start
```sh
yarn dev
```
