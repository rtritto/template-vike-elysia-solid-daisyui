# template-vike-solid-daisyui

Template stack:
- **Vike**
- **Vite**
- **Solid**
- **DaisyUI**
- **ESLint**
- **TypeScript**

## Clone
`git clone https://github.com/rtritto/template-vite-solid-daisyui.git`

## Install
`yarn`

## Start
`yarn start`

OR

`yarn dev`
